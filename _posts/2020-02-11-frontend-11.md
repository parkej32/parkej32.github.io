# Vue Loader

.vue 형태의 파일을 javascript 모듈 형태로 변환해주는 webpack loader.

## vue-loader 기능들

- ES6지원
- `<style>`, `<template>` 에 각각에 다른 webpack 로더를 지원.
- 각 .vue 컴포넌트 스코프의 css 스타일링 지원
- webpack의 모듈 번들링 지원과 의존성 괸리 제공.
- 개발 시 Hot Reloading 지원.
