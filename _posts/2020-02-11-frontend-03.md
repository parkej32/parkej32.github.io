# VUE 구조

.vue 단위로 template, script, style 세개의 영역을 갖음.  
template 영역에 html(View)을 작성하고, script(Model)는 뷰에서 사용할 data 및 component, 함수들을 작성.  
style 에는 View단의 표현을 구현하는데, 해당 .vue 영역에만 스타일을 적용하고 싶을 경우 scoped 속성을 사용.

```html
<template> </template>

<script></script>

<style scoped></style>
```
